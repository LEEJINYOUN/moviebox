# 영화 웹 사이트

<h2> 📃 개요 </h2>

- <h3>현재 상영하거나 상영예정인 영화리스트 보기</h3>
- <h3>관람한 영화의 관람평 작성</h3>
  <br/><br/>

<h2> 🛠 사용 기술 </h2>

- <h3>React</h3>
- <h3>Firebase</h3>
  <br/><br/>

<h2> ⚙️ 구현 기능 </h2>

- <h3>회원가입, 로그인 (구글, 이메일) 기능</h3>
- <h3>관람평 작성 (로그인한 후에 가능) 기능</h3>
- <h3>관람평 수정 및 삭제 기능</h3>
  <br/><br/>
